# 🌱 GreenGuide – AI-Based Sustainable Living Android Application

## 📌 Project Overview
GreenGuide is an AI-powered Android application designed to help users with plant identification, plant disease detection, and smart plant care guidance.  
The system integrates mobile technology, artificial intelligence, and cloud backend services to promote sustainable and informed plant care.

---

## 🎯 Objectives
- Identify plants using AI-based image analysis  
- Detect plant diseases and provide remedies  
- Enable smart plant care with reminders and growth tracking  
- Provide a user-friendly Android mobile interface  

---

## 🛠 Technologies Used
- Android Studio
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore & Cloud Storage
- Python (Flask / FastAPI)
- TensorFlow / CNN
- GitHub

---

## 👥 Team Roles & Responsibilities

### 🔹 Member 1 – Frontend & Mobile Application Developer
- Developed Android application using Kotlin and Jetpack Compose  
- Implemented Firebase Authentication (Email & Google Sign-In)  
- Integrated camera and gallery for plant & disease detection  
- Designed UI navigation and ensured smooth user interaction  

### 🔹 Member 2 – AI / Machine Learning Engineer
- Collected and preprocessed plant and disease image datasets  
- Developed CNN models for plant identification and disease detection  
- Implemented multi-image fusion for better accuracy  
- Evaluated and optimized model performance  

### 🔹 Member 3 – Backend & Database Engineer & GitHub Maintainer
- Developed backend APIs using Python (Flask/FastAPI)  
- Integrated trained AI models with backend services  
- Designed and managed Firebase database and cloud storage  
- Implemented smart care logic (watering schedules, reminders)  
- Managed GitHub repository, branches, and pull requests  

### 🔹 Member 4 – Documentation, Testing & Presentation Lead
- Prepared complete project documentation and report  
- Performed functional testing and validation  
- Prepared PPT, diagrams, and demo explanation  
- Coordinated viva preparation and final presentation
